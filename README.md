# Connect Four (Godot 4.6)

This project was a fun experiment using GitHub Copilot to build a fully playable Connect Four game in Godot. Every design decision, feature, and line of code was driven entirely by prompts — see [docs/project-prompts.md](docs/project-prompts.md) for the full log of prompts used to create this project.

## How to Play

1. Download and install **Godot 4.6** (or 4.6.1) from [godotengine.org/download](https://godotengine.org/download/).
2. Download or clone this repository to your local machine.
3. Open Godot, click **Import**, and select `project.godot` in this folder.
4. Open the imported project and press `F5` (or click **Play**) to run the game.
5. In the setup screen, choose Human or Computer for each player, then select difficulty for any Computer player.
6. Start the match and click a column to drop a disc. Connect four in a row to win.
