# Connect Four (Godot 4.6)

This project was a fun experiment using GitHub Copilot to build a fully playable Connect Four game in Godot.

## How to Play

1. Download and install Godot 4.6 (or 4.6.1) from the official Godot website.
2. Download or clone this repository to your local machine.
3. Open Godot, click **Import**, and select `project.godot` in this folder.
4. Open the imported project and press `F5` (or click **Play**) to run the game.
5. In the setup screen, choose Human or Computer for each player, then select difficulty for any Computer player.
6. Start the match and click a column to drop a disc. Connect four in a row to win.
